# 🎯 Guess Master – The Ultimate Number Challenge

A modern, animated number guessing game built with **React, TypeScript, Tailwind CSS, and Framer Motion**.

Guess the secret number between **1 and 100** in the fewest attempts possible and try to beat your personal best!

---

## 🚀 Live Features

- 🎮 Random number generation (1–100)
- 📊 Attempt counter
- 🏆 Personal Best (saved using LocalStorage)
- 💥 Victory screen with ranking (S / A / B)
- 📜 Guess history timeline
- 🎨 Modern glassmorphism UI
- ✨ Smooth animations using Framer Motion
- 📱 Fully responsive layout
- ⚡ Single-screen compact design

---

## 🛠️ Tech Stack

- **React**
- **TypeScript**
- **Tailwind CSS**
- **Framer Motion**
- **Lucide Icons**

## 🎮 How to Play

1. Enter a number between **1 and 100**
2. The game will tell you:
   - Too Low 📉
   - Too High 📈
3. Keep guessing until you find the correct number
4. Try to win in the fewest attempts!

### 🏅 Ranking System

| Attempts | Rank |
|----------|------|
| 1–5      | S    |
| 6–8      | A    |
| 9+       | B    |

---


1. Install dependencies:
   `npm install`
2. 
   `npm run dev`
